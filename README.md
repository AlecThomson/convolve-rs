# Convolve-rs

A port of `beamcon` from [RACS-tools](https://github.com/alecthomson/RACS-tools) from Python to Rust. Also relies on approaches from [Miriad](https://github.com/csiro/miriad).

NOTE: This is an experiment with LLM-assisted coding with Claude. Do not trust this software as far as you can throw it.
